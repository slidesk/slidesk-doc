# Manage plugins

:::note
(since 2.7.0)
:::

SliDesk has a repository where you can find extra plugins https://github.com/slidesk/slidesk-extras.

A command allows you to install plugins from this repository:

```sh
slidesk plugin install <name>
```

You have 4 actions available for plugins: `install`, `update`, `remove` and `list`.

## List all plugins availables in the repository

```sh
slidesk plugin list
```
